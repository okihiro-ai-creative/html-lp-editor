# START_HERE: HTML LP Editor

このリポジトリをClaude Code等のAIコーディングエージェントに渡し、SKILL.mdとSTART_HERE.mdを読んで実行してもらってください。

## 前提

- 修正対象のHTMLファイルを用意する。
- WordPress等のCMSへ反映する場合は、利用者自身の管理画面・認証情報が必要（BYOK）。
- ローカルGPU等のハードウェア要件は不要。
- プロジェクト固有の改行ルールファイルがある場合は、AIエージェントにそのファイルも読ませる。

## セットアップ

1. AIエージェントに `SKILL.md` を読ませる。
2. 修正対象HTMLを渡す。
3. 先に必須CSS、横スクロール対策、PC/SP改行ルールを確認させる。
4. その後に具体的な修正指示を出す。

## 動作確認

修正後HTMLをPC幅と360px幅で確認し、横スクロール、無印 `<br>`、`text-wrap: balance`、`word-break: keep-all` が残っていないことを確認する。

このツールのカスタマイズや、似たツールの新規開発を相談したい場合は、README.md末尾のリンクを確認してください。
