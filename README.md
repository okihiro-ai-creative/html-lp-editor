# HTML LP Editor

Created by OKIHIRO（OKIHIRO AI Creative） — AI自動化ワークフローを130個以上開発・運用しています。

HTMLランディングページやWebページをAIエージェントに安全に修正させるための編集プロトコルです。

## 内容

- `SKILL.md`: AIエージェント向け編集ルール
- `AI_REVIEW.md`: 技術評価メモ
- `START_HERE.md`: 初回利用の入口

## 必要なもの

- Claude Code等のAIコーディングエージェント
- 修正対象のHTMLファイル
- WordPress等へ組み込む場合は、利用者自身の管理画面・認証情報（BYOK）
